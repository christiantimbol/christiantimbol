Summary

Made a Twitter retweet bot to keep you in the loop about what's going viral on Twitter, built with Python 3 and Tweepy. Tweets with retweet_count exceeding 30,000 (originally 10,000) are retweeted by @ViralityBot when run.

Please see ViralityBot's Twitter page for the outcome of the Python 3 script (as well as the "viral" tweets), and also see ViralityBot's project page on GitHub for more information.
